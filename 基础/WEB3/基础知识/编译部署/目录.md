# Solc
> 这个是编译solidity的


安装(最好安装相同的版本)： 
```sh
npm install -g solc
```

```sh
npm install solc@0.8.20
```

使用：
```sh
solcjs --bin --include-path node_modules/ --base-path . -o . MainContract.sol
```
