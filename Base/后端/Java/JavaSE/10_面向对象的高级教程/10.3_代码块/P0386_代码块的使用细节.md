## 1. 代码块的基本介绍：

代码化块又称为初始块,属于类中的成员[即 是类的一部分]，类似于方法，将逻辑语句封装在方法体中，通过{}包围起来。
但和方法不同，**没有方法名，没有返回，没有参数，只有方法体，而且不用通过对象或类显式调用，而是加载类时，或创建对象时隐式调用**



## 2. 基本语法：

[修饰符] {

代码

}
